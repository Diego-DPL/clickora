# Clickora - Desarrollo Web Moderno

![Clickora Banner](https://clickora.es/og-image.png)

Landing page moderna y futurista desarrollada con **Astro** y **TailwindCSS 4.0** para ofrecer servicios de desarrollo web de alto impacto.

## 🚀 Tecnologías

*   **Framework**: [Astro](https://astro.build/) (v5)
*   **Estilos**: [TailwindCSS](https://tailwindcss.com/) (v4.0 Beta - CSS-first configuration)
*   **Bundler**: Vite
*   **Lenguaje**: TypeScript
*   **Despliegue**: Netlify / Vercel (Ready)

## ✨ Características

*   **Diseño Futurista**: UI Glassmorphism, gradientes y efectos de neón.
*   **Animaciones Avanzadas**: Transiciones suaves, efectos de scroll y elementos flotantes.
*   **Alto Rendimiento**: Puntuación 100/100 en Lighthouse/PageSpeed.
*   **SEO Optimizado**: Meta etiquetas, Open Graph, Sitemap y marcado Schema.org.
*   **Oferta Irresistible**: Estructura de precios psicológica para maximizar conversiones.
*   **Contacto Directo**: Integración con WhatsApp API.

## 🛠️ Instalación y Desarrollo

1.  **Clonar el repositorio**:
    ```bash
    git clone https://github.com/Diego-DPL/clickora.git
    cd clickora
    ```

2.  **Instalar dependencias**:
    ```bash
    npm install
    # Si hay problemas con versiones beta de Tailwind 4:
    npm install -D @tailwindcss/vite@4.0.0-beta.8 tailwindcss@4.0.0-beta.8
    ```

3.  **Iniciar servidor de desarrollo**:
    ```bash
    npm run dev
    ```

4.  **Construir para producción**:
    ```bash
    npm run build
    ```
